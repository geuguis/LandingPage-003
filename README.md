## ♠️♦️❤️♣️ Landing Page José Luis - Balatro Edition ♠️♦️❤️♣️

Landing page temática diseñada para un desarrollador **Backend**, inspirada al 100% en la estética del juego **Balatro**. Mezcla el estilo visual de las "Joker Cards" con un stack técnico de Java y Spring Boot.

##  Lo que hace especial a este repositorio:
- **Estética Balatro:** Uso de cajas de descripción tipo "tooltip", fuentes pixel art y paleta de colores del juego.
- **Efectos Visuales:** Background dinámico mediante un shader de **WebGL** y líneas de escaneo (scanlines) para ese toque retro/CRT [inspirado en este repositorio](https://github.com/Azkun/balatroShader).
- **Interactividad:** Efecto de inclinación 3D (tilt) en las cartas al pasar el ratón y animaciones de "bounce" al hacer clic, imitando el feedback del juego.
- **Secciones Temáticas:**
    - **Shop:** Las habilidades se muestran como cartas en la tienda del juego.
    - **Journey:** Línea de tiempo con estilo de "run" de Balatro.
    - **Passion:** Integración de elementos musicales (trombón) con notas flotantes en JS.

## Tecnologías
- **Frontend:** HTML5, CSS3 (Variables personalizadas y animaciones `@keyframes`).
- **Lógica:** Vanilla JavaScript (para los efectos de movimiento 3D y partículas).
- **Gráficos:** WebGL Shader (para el fondo hipnótico).
- **Fuentes:** Google Fonts + fuentes externas personalizadas.

##  Uso
No necesita compilación ni servidores complejos. Es una landing estática:
1. Clona el repo.
2. Abre el `index.html` en tu navegador.
